Aturan Validitas

  1. Panjang input harus tepat 10 digit.
  2. Digit pertama tidak boleh 0 (kode bank/cabang).
  3. Digit ke-2 hingga ke-10 bebas 0–9.
  4. Karakter selain digit (huruf, spasi, simbol) langsung menuju state jebakan qtrap.
